# RUN.exe

## About This Project
This game was developed as part of my Final Year Project (FYP).

## Installation & Playtest

1. Download the latest build from the Releases section of this repository.
2. Extract the downloaded ZIP file to your preferred location.
3. Open the extracted folder.
4. Run the `.exe` file to start the game.
